# dotDashboard

Web App Dashboard สำหรับโปรเจกต์ dotWatch / IoT Device Monitoring

## Run บนเครื่อง

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy GitHub Pages

1. สร้าง Repository ชื่อ `dotDashboard`
2. Push โปรเจกต์นี้ขึ้น GitHub
3. ไปที่ `Settings > Pages`
4. เลือก Source เป็น `GitHub Actions`
5. URL จะเป็น:

```txt
https://YOUR_USERNAME.github.io/dotDashboard/
```

## Firebase

คัดลอกไฟล์ `.env.example` เป็น `.env` แล้วใส่ค่าจาก Firebase Project ของคุณ
